# Clubhouse

Clubhouse helps manage student organizations.

[![Travis CI][1]][2]


## Getting Started

Install the system services Clubhouse relies on:

* MySQL 5.6

Clone the Clubhouse repository and run the setup script:

```bash
git clone git@github.com:Pitt-CSC/clubhouse.git
cd clubhouse
bin/setup
```

That's it!
